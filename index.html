4. 'feature' and 'layer' are both attributes of the data in the geojson dataset. 'layer' seems to be a container for 'feature', as it is contained within it, along with a number of other attributes which control when and how the data is displayed. In 'feature', we find 'geometry', which dictates what kind of spatial data this is (in this case, point), 'type', which in this case is always 'Feature', and 'properties', which contains an object containing any other attributes/metadata about the given datapoint (such as addresses and station types).
